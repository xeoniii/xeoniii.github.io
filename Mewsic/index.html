Mewsic

v0.9.6 GPL v3.0 Beta Cross-Platform Desktop

Mewsic is a modern, performance-driven offline music player built for the modern desktop. It combines the speed and safety of Rust with the flexibility of React to deliver a seamless listening experience.

Plugin API Documentation

Built for Speed

Leveraging Rust's performance for audio processing and file management, ensuring a lag-free experience even with massive libraries.

Modern UI

A clean, Material Design inspired interface that stays out of your way and lets your music take center stage.
